Widget Lab

Test dashboard widgets locally — no worker, no auth, no server. Pick a widget.

<iot-color>
HSV colour wheel — pick a colour, send hex / hsv / rgb.
<iot-slider>
Numeric value within a range; horizontal or vertical.
<iot-toggle>
On/off command; reflects reported state.
<iot-value>
Read-only numeric/text readout with unit.
<iot-gauge>
Radial gauge over a min/max range.
<iot-percent>
Percentage ring with colour thresholds.
<iot-push>
One-shot momentary command button.
<iot-chart>
Multi-series time-series chart (ApexCharts, via CDN).
<iot-map>
Geographic markers on a Leaflet map (via CDN).
